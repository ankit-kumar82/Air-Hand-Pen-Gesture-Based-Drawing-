# ✋ Air Hand Pen – Gesture Based Drawing System

**Air-Hand-Pen-Gesture-Based-Drawing** is a computer vision-based project that allows users to draw in the air using hand gestures without touching any physical device.
It uses real-time hand tracking to convert finger movements into digital drawings on the screen.

---

## 🚀 Features

* ✋ Real-time **hand gesture recognition**
* 🖊️ Draw in air using **index finger**
* 🎨 Change drawing **color dynamically**
* 📏 Adjustable **brush size**
* 🧹 **Clear screen** using open palm gesture
* ⏸️ **Pause drawing** using fist gesture
* ⚡ Smooth and responsive drawing experience

## ⚙️ Working Principle

1. Webcam captures live video input
2. **MediaPipe Hands** detects hand landmarks
3. Tracks the **index finger tip position**
4. Converts movement into drawing on canvas
5. Recognizes gestures to control actions

---

## 🧠 Gesture Controls

| Gesture            | Action       |
| ------------------ | ------------ |
| 👉 Index Finger Up | Draw         |
| ✊ Closed Fist      | Pause        |
| 🖐️ Open Palm      | Clear Screen |

---

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Run:

   ```
   index.html
   ```
4. Allow camera permission in browser
5. Start drawing in air ✨
